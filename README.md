# Personal Portfolio Website

## Project Overview

This project is a personal portfolio website created for CSCI 355: Internet and Web Technologies. The purpose of the website is to provide a professional online presence that showcases my background, technical skills, projects, and career goals.

The website was built using HTML, CSS, and JavaScript and is hosted using GitHub Pages.
